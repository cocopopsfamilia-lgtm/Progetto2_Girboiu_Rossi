    Sito web realizzato con HTML, CSS e JavaScript che punta a promuovere e raccontare il progetto ERASMUS all'interno dell'Istituto Tecnico e Tecnologico "Allievi Sangallo" di Terni.

# Funzionalità del Sito
    - Layout Moderno (Glassmorphism): Utilizzo di paragrafi con effetto "vetro sfocato" per un'interfaccia più futuristica (presa ispirazione dall'ultimo aggiornamento dei sistemi Apple).
    - Slider di Sfondo Dinamico (per Home Page): Uno slideshow JavaScript che cambia le immagini (che si trovano all'interno di un array) di sfondo in modo fluido grazie a transizioni CSS.
    - Layout Responsivo: Utilizzo della **Flexbox** che permette la corretta visualizzazione sia su PC che su smartphone (con immagini che si auto-adattano).
    - Sezione Internazionale: Spazio dedicato all'impatto dell'Erasmus+ sulle competenze degli studenti (PCTO, lingue, lavori, ecc.).

# Linguaggi utilizzati
    - HTML: Utilizzato per la strutturazione del sito e di tutti i contenuti della pagina
    - JavaScript: Utilizzato per la logica delle azioni che vengono svolte dal sito (es: slideshow)
    - CSS: Utilizzato per la parte di interfaccia grafica del sito (che comprende tutti gli stili delle scritte, paragrafi trasparenti, ecc.)
